## How To Play

1.  Clone or Download this repo

```
git clone https://github.com/3pic/curly-War.git
```

2.  run (dadadadadadada ...)

<img src="https://media.giphy.com/media/7kn27lnYSAE9O/giphy.gif" width="30%">

* Linux/MacOs:
```
chmod u+x run.sh
./run.sh
```

* Windows:
```
run
```
